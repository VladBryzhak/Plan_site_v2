# Мій план тренувань — v2

Персональний веб-додаток (PWA) із планом тренувань та харчування, розрахований приблизно на 12 тижнів. Мета — поступове схуднення з 90 до 82 кг за збалансованою програмою силових, кардіо- та відновлювальних днів, разом із продуманим раціоном.

## Огляд

Додаток повністю клієнтський: написаний на чистих HTML, CSS та JavaScript, без фреймворків і без бекенду. Усі дані користувача (відмітки виконаних тренувань, обрана тема) зберігаються локально в браузері через localStorage, тож додаток працює автономно й не потребує сервера. Інтерфейс організований у три вкладки: Тренування, Харчування та Поради.

## Основні можливості

- Тижневий розклад — дні з різними типами навантаження (верхня частина A/B, кардіо + кор, ноги, пробіжка, відпочинок) із кольоровими позначками.
- Відео техніки виконання — для кожної вправи доступний акордеон, що відкриває випадково обране YouTube-відео та текстову підказку з техніки.
- Відмітка тренувань — кнопка «Виконано» з лічильником у шапці та прив'язкою до конкретної дати тижня.
- Темна тема — перемикач вручну та автоматичне визначення системної теми.
- План харчування — раціон із перемиканням між тижнями (без яєць, ~2000–2100 ккал, 155–170 г білка, 5 прийомів їжі на день).
- Експорт у календар — генерація запрошення .ics для додавання тренувань у календар.
- Офлайн-режим — Service Worker кешує всі основні файли для роботи без інтернету.
- PWA — встановлюється на домашній екран (зокрема на iPhone) у standalone-режимі з власними іконками.

## Технології

- HTML5 / CSS3 — розмітка та адаптивні стилі (світла та темна тема).
- Vanilla JavaScript — уся логіка без зовнішніх залежностей.
- PWA — manifest.json та Service Worker (sw.js).
- localStorage — збереження стану на пристрої користувача.
- Google Fonts — DM Sans та DM Serif Display.
- Python — допоміжний скрипт генерації іконок.

## Структура проєкту

- index.html — розмітка та структура сторінки
- manifest.json — PWA маніфест
- sw.js — Service Worker (офлайн-кеш)
- generate_icons.py — скрипт генерації іконок
- css/styles.css — стилі (світла та темна тема)
- js/data.js — дані: розклад, вправи, відео, харчування
- js/app.js — логіка додатку
- icons/ — іконки для PWA

## Як запустити локально

Відкрийте index.html у браузері. Зверніть увагу, що Service Worker (офлайн-режим та встановлення PWA) працює лише через HTTPS або localhost.
